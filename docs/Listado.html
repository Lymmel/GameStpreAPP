Menú ayuda

  • En la actividad Listado mostramos la lista de elementos guardada en nuestra base de datos

  • Cada elemento de esa lista tiene 2 opciones:

    • Swipe de derecha a izquierda para borrar el elemento.

    • Swipe de izquierda a derecha para editar el elemento.

  • Botón flotante para añadir otro elemento a la lista.
